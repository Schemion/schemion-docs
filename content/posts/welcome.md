# Добро пожаловать в Schemion Docs

Документация Schemion теперь разложена по рабочим разделам: архитектура, API, данные, эксплуатация и разработка. Так проще быстро найти нужный слой системы и не держать все детали в одном длинном файле.

Если вы впервые открыли проект, начните с [главной страницы](/), затем перейдите в [документацию](/docs/). Для запуска окружения пригодится раздел [эксплуатации](/docs/operations/), а для интеграций и клиентского кода - [API reference](/docs/api/reference/).

Дальше базу знаний можно расширять по мере развития платформы: добавлять рецепты, описывать новые архитектуры моделей, фиксировать решения по инфраструктуре и собирать частые ошибки в отдельный troubleshooting-раздел.
