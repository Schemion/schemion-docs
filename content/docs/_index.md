# Документация

- [Архитектура](/docs/architecture/) - компоненты платформы и потоки задач.
- [API](/docs/api/) - быстрый обзор и полная справка по эндпоинтам.
- [Данные](/docs/data/) - датасеты, PostgreSQL, MinIO и кэш.
- [Эксплуатация](/docs/operations/) - Docker Compose, окружение, порты и ограничения.
- [Разработка](/docs/development/) - локальная работа, тесты и расширение сервисов.
